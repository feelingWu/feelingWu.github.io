官方网站：

https://hexo.io/zh-cn/docs/configuration

清除缓存文件 (db.json) 和已生成的静态文件 (public)

$ hexo clean

生成静态文件

$ hexo generate
-d, –deploy 文件生成后立即部署网站
-w, –watch 监视文件变动

部署

hexo d

启动服务器。默认情况下,访问网址为: http://localhost:4000/

$ hexo server
-p, –port 重设端口
-s, –static 只使用静态文件
-l, –log 启动日记记录，使用覆盖记录格式

发表草稿

$ hexo publish [layout]

显示草稿

$ hexo –draft

自定义 CWD（Current working directory）

$ hexo –cwd /path/to/cwd

hexo new [layout] <br>ex:<br> hexo new study demo01

hexo publish [layout] <br>ex:<br> hexo new draft demo01<br> hexo publish study <title>
